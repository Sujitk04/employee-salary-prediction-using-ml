# 💼 Employee Salary Prediction using ML

🚀 A full-stack Machine Learning project that predicts whether an employee earns **>50K or ≤50K salary** based on features like age, education, occupation, and more.

---

## 🛠️ Tech Stack
**Frontend:** React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**ML:** Python (Pandas, NumPy, Scikit-learn, Jupyter)

---

## ✨ Features
✔️ User Login & Register  
✔️ Salary Prediction System   
✔️ Data Visualization (Bar & Pie Charts)  
✔️ Model Accuracy Display  
✔️ Full Stack Integration  

---

## 🧠 ML Model
- Algorithm: Logistic Regression / Random Forest  
- Preprocessing: Encoding, Scaling, Missing Value Handling  
- Evaluation: Accuracy Score, Confusion Matrix
